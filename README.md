# Salus
Protect yourself today with Salus, the personal security app that not only places a danger alarm in your smartphone but dials the emergency number when danger is imminent. 
This app currently only supports the Malaysia emergency number.
When in danger press the Send SOS option and the Pocket Security App will send an SMS to your emergency number along with your current exact location and some other information that will be very useful for the receiver to trace you faster. 
The App finds your location using GPS. 
Other than this the app also provides an alarm to warn people in case of danger and calling Police facility. 
Stay Safe Everywhere!
